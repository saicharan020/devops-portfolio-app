About this project

This is my DevOps portfolio website. I built it as a static site (HTML/CSS), containerized it with Docker, automated image builds & pushes with GitHub Actions, and deployed it on Azure App Service (Web App for Containers). Previously, I also hosted the static site via GitHub Pages.

End-to-end journey (what I did)

  1. Built a simple static website using HTML & CSS.
  2. Wrote a Dockerfile and containerized the site with Docker.
  3. Pushed the image to Docker Hub.
  4. Created a GitHub Actions workflow (YAML) to automatically build & push the Docker image on every push to main.
  5. Initially hosted the static site using GitHub Pages to make it publicly available.
  6. Deployed the Docker image to Azure Web App for Containers so it runs 24/7.
  7. (Optional/Planned) Enabled Continuous Deployment from Docker Hub to Azure using a webhook.

What this demonstrates

  • Version control with Git & GitHub
  • Containerization with Docker
  • CI/CD automation using GitHub Actions
  • Image publishing via Docker Hub
  • Cloud deployment on Azure App Service
  • Static hosting via GitHub Pages
